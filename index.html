BassPlayer

An Audio Player based on bass.dll components & a Total Commander lister plugin.

Project intended to be a small & efficient replacement for Winamp 2.x series users.
This means that the program hasn't got an internal media library, but supports iTunes as an external media library

Compiled binary can be downloaded from: My Google Drive

Features

  • Supports various audio formats:
    MP3, MP4, AAC, OGG, WAV, FLAC, WavPack, Apple Lossless (ALAC), WMA, AC3, CDDA
  • Supports various playlist formats:
    PLS, M3U, WPL (Windows Media Player playlists), BPL (Player's native playlist format)
  • 32 bit and 64 bit support
  • Integrated file browser
  • Supports browsing of iTunes database (iTunes required)
  • Supports Windows 7 & 8 Shell features: Taskbar progress & Taskbar preview buttons
  • Supports Keyboard media keys
  • Supports streaming playback from Youtube

Required tools to compile

  • Visual Studio Community edition
  • Bass.net API -> http://bass.radio42.com/

Requirements to run

  • Windows 7 or Windows 8 or Windows 8.1 or Windows 10
  • .NET Framework 4.5 -> Only needed on Windows 7
